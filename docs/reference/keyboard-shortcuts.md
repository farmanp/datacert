# Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `Ctrl/Cmd + K` | Focus search | Profile view |
| `Esc` | Close modal/dialog | Any modal open |
| `?` | Show keyboard shortcuts | Any page |

---

## Profile View

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + F` | Focus column search |
| `T` | Switch to Table View |
| `C` | Switch to Card View |
| `Q` | Switch to Quality tab |
| `E` | Open Export dialog |
| `R` | Recompute correlation matrix |

---

## SQL Mode

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + Enter` | Run query |
| `Ctrl/Cmd + /` | Toggle comment |
| `Tab` | Indent selection |
| `Shift + Tab` | Outdent selection |
| `Ctrl/Cmd + L` | Select line |

---

## Export Dialog

| Shortcut | Action |
|----------|--------|
| `1-7` | Select export format (1=HTML, 2=JSON, ...) |
| `Enter` | Confirm and download |
| `Esc` | Close dialog |

---

## Navigation

| Shortcut | Action |
|----------|--------|
| `G H` | Go to Home |
| `G S` | Go to SQL Mode |
| `Alt + ‚Üê` | Browser back |

---

## Accessibility

| Shortcut | Action |
|----------|--------|
| `Tab` | Navigate forward through interactive elements |
| `Shift + Tab` | Navigate backward |
| `Enter/Space` | Activate button/link |
| `Arrow keys` | Navigate table cells |

---

## Browser Defaults

*DataCert preserves standard browser shortcuts:*

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + R` | Reload page |
| `Ctrl/Cmd + W` | Close tab |
| `Ctrl/Cmd + T` | New tab |
| `Ctrl/Cmd + Shift + I` | Open DevTools |
| `F12` | Open DevTools |

---

## Tips

- **Vim users:** No Vim mode yet, but `hjkl` navigation coming soon
- **Screen readers:** All shortcuts announced via ARIA labels
- **Customization:** Shortcuts currently not customizable (roadmap feature)
